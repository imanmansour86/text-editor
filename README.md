## Installation

- Go to the app's [repo](https://github.com/imanmansour86/text-editor) in github and clone the app

- From terminal: navigate to app's directoty and run:

  ```md
  $ npm install
  ```

- To invoke the app from terminal, run:

  ```md
  $ npm run start
  ```
